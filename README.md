# react-native-xd5-405-label-printer

react-native plugin for the bluetooth XD5-40t printer

## Installation

```sh
yarn add react-native-xd5-405-label-printer
```

## Usage

```js
import Xd5405LabelPrinter from "react-native-xd5-405-label-printer";

// ...


```
